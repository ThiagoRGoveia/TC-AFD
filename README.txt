Windows:
Para executar o programa em Windows x64, inicie o arquivo 'index-win.exe'.
Será gerado um 'output.txt', dado um 'input.txt'.

Outros OSs:
Para executar o programa, inicie o arquivo 'index-[os]', aonde [os] é o seu sistema operacional.
Também será gerado um 'output.txt', dado um 'input.txt'.
